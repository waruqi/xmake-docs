# Multi-level Directories

We can use the `includes()` interface to add sub-project files to achieve multi-level directory configuration.

For a detailed tutorial, see [Multi-Level Directories Guide](/guide/project-configuration/multi-level-directories). For namespace isolation, see [Namespace Isolation](/guide/project-configuration/namespace-isolation).
