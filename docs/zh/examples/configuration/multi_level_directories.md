# 多级目录配置

我们可以使用 `includes()` 接口来添加子工程文件，实现多级目录配置。

多级目录配置的详细教程请参阅[多级配置指南](/zh/guide/project-configuration/multi-level-directories)。关于命名空间隔离请参阅[命名空间隔离](/zh/guide/project-configuration/namespace-isolation)。
