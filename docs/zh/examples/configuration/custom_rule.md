# 自定义构建规则

我们可以使用 `rule()` 定义自定义构建规则，来处理特定的文件类型或构建逻辑。

自定义规则的入门教程请参阅[自定义规则指南](/zh/guide/project-configuration/custom-rule)。完整的规则描述域接口请参阅[自定义规则 API](/zh/api/description/custom-rule)。内置规则列表请参阅[内置规则](/zh/api/description/builtin-rules)。
